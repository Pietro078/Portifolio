_Sobre mim_

Comecei na programação por curiosidade. Fiquei alguns meses estudando logica de programação, o que eram as linguagens orientadas a objetos, o que era a internet e como nós poderiamos upar um site para a "nuvem" e faze-lo ser visto por qualquer pessoa.

Minha primeira linguagem de programação foi python por ser facil de expressar os comandos para a maquina. Logo depois fui para javaScript para aprender a integra-lós ao html e css.

No momento estou no segundo semestre da faculdade Impacta cursando ADS. Na faculdade estão ensinando mais sobre POO com python então estou aproveitando para estudar o framework django e usar o worm do peewee para todas as funcionalidades basicas de um banco de dados.
